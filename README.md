## 共建小白用的JavaScript技术文档

1. 如果你发现文档有错误，欢迎提交pull request，不胜感激。
2. 如果你有好的总结，请提交pull request，并在文档的内容中注明你的身份。
3. 该项目每天都在更新，在你进行提交之前，请先git pull一下，拉取最新的内容。

## 提交的文档review方式

1. 必须使用md格式。
2. 图片墙必须使用github，不允许使用其他任何图片墙。
2. 文档尽量能让小白看懂，特别是注意事项要清楚。
